# HOMEHUB FRONTEND

## INSTALLATIONS 

### 1. React and Vite configurations

```bash
npm create vite@latest homehub-frontend
```
--- 

### 2. Tailwind Configuration
```bash
npm install tailwindcss @tailwindcss/vite
```
---

### 3. Shadcn Components
- Initialization 
```bash 
npx shadcn@latest init
```
- Installations
```bash 
npx shadcn@latest add sonner button input select dialog badge card
```
